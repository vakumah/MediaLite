# üé¨ MediaLite

**Compress & enhance your media**

All-in-one tool for compressing videos and photos while making them look sharper and more vibrant. Target output under 1MB with HD quality enhancement.

## Features

### Video Compression
- Smart bitrate calculation for target file size
- Sharpening filter for crisp output
- Color enhancement (contrast + saturation boost)
- Resolution options: 720p, 480p, 360p, 240p
- HD color preservation (BT.709 color space)
- Audio removal option

### Photo Processing
- Support all formats including HEIC (iPhone Live Photos)
- Sharpness enhancement with adjustable intensity
- Contrast and color boost
- Smart compression to target size
- Before/after comparison preview

## Supported Formats

| Type | Input Formats | Output |
|------|--------------|--------|
| Video | MP4, MOV, MKV, AVI, WebM, 3GP | MP4 (H.264) |
| Photo | JPG, PNG, WebP, HEIC, HEIF, BMP, TIFF, GIF | JPEG |

## Tech Stack

- **Frontend**: Streamlit
- **Video Engine**: FFmpeg
- **Image Engine**: Pillow
